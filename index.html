Overview

NextGen Extras is an enhanced 'extras' experience around user owned content. This reference library is built to allow implementators to accelerate their NextGen Extras iPad Tablet implementation.

Data binding + UI layers

The reference code for iPad & Android Tablets is being built to be modular; e.g. a partner retailer can choose to use just the manifest-to-native data-binding layer and/or the UI layer. Each layer is customizable by retailer is they choose to do so. Architectural Layers

Podfile

To integrate NextGen Extras into your Xcode project using CocoaPods, specify it in your Podfile:

pod 'NextGenExtras'
